UIHostingView

public class UIHostingView : UIView

A UIKit view that hosts a ArgoKit view hierarchy.

Example:

         let hostView: UIHostingView = UIHostingView(content: Content(), frame:self.view.bounds)
         self.view.addSubview(hostView)
